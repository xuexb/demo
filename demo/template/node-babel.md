# node端测试package模板-babel


- fecs - 代码检查
- mocha - 单元测试
- chai - 断言
- sinon - 测试桩
- sinon-chai
- istanbul - 代码覆盖率
- babel - 编译器

```json
{

    "test:watch": "npm run test -- --watch",
    "test:cov": "istanbul cover node_modules/mocha/bin/_mocha -- --compilers js:babel-register -t 5000 --recursive  -R spec test/",
```
